# TypeScript LeetCode Exercises

Este repositorio contiene ejercicios de LeetCode resueltos (y/o enunciados) usando TypeScript. Está pensado como un cuaderno de práctica personal donde cada problema tiene su propia carpeta con el enunciado y la solución.

## Estructura

Cada ejercicio vive en su propia carpeta con el formato:

- `NNNN. Nombre Del Problema/`
	- `exercise.md` — enunciado del problema (input/output, ejemplos).
	- `solution.ts` — implementación en TypeScript.

Ejemplo:

```
2704. To Be Or Not To Be/
	exercise.md
	solution.ts
```

## Convenciones

- Uso de TypeScript para las soluciones cuando procede.
- Nombre de carpetas con número de problema seguido del título para facilitar búsqueda.
- Cada `exercise.md` contiene el enunciado y ejemplos de entrada/salida.

## Contribuciones

Si quieres añadir soluciones o enunciados, crea una carpeta nueva siguiendo la convención de nombres y añade `exercise.md` y/o `solution.ts`. Abre un pull request con una breve descripción.
